About this website

This website is an IS 117 assignment for NJIT professor Keith Williams. Each of the chapters is writing done by me to summarize videos that we have watched. Definitely do not recommend this site as a good source of information about anything really. This is the first fully fleshed out website we have done, how exciting! I hope you enjoy it as much as I cried making it (just kidding bootstrap made this super easy).

1. At least 20 commits and 9 branches(approx 1 branch per page)

2. Commits labeled Bug, Feature, Task i.e. Feature: New Page, Bug: fixed header, Task: created gitignore

3. Both partners need to submit.

4.At least 1 picture per page from a legal online source:http://www.pexels.com/ or http://www.unsplash.com/

5.You need about 1-2 pages of text per page about each of the videos, so you are required to submit a word document that includes the text. You should basically create a mockup of your page in word and then complete it in Bootstrap.
