W3C and HTML and CSS

Who is W3C?

W3C stands for the World Wide Web Consortium and they are an organization dedicated towards improving the Web through the creation and development of guidelines, standards, and protocols, such as HTML and CSS. They are led by Tim Berners-Lee, the inventor of the World Wide Web. For more about W3C, visit their website.

What is HTML?

HTML is a markup language used to describe the structure of a webpage. Elements of a webpage such as "paragraph" or "header" are used to label specific parts of the page. For example, the term "aside" is used to describe an element of a page that is set to the side of a page, which is usually the right side for pages in right-to-left languages. If you are using the Google Chrome or Mozilla Firefox web browsers on a PC, you can press f12 to look at the HTML code for this webpage yourself.

What is CSS?

CSS is a language used to describe how a webpage should be presented. There are various properties that can be used to stylize a page.

For example, you could:

  • change the color of the text on a page
  • change the background of an element
  • add or remove borders to an element
  • or any combination of these and other properties.
